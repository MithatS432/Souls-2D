# Souls-2D
Escaping ghosts, surviving and trying to move forward.

# Game Description
The spirits, once protected by three guardians, have now been shattered, transforming the three guardians into different forms and inciting them against their own people.It was up to us to gather and unite these souls.

Our goal is to kill all enemies, defeat 3 boss enemies, unite all souls, gain experience, and level up.

# Controller
To walk, use w, a, s, d or the arrow keys

To run faster, LeftShift

Space key to jump 

To jump twice, press the Space key twice

Press Fire1 to attack

Tab key for inventory

Dashing with the R key

Magic attack with the F key

# Things i have arranged in the game system
-Heredity + Activity-based extension

-The boss inherits the enemy's behavior

-The "die()" method works polymorphically

-Health changes are broadcast via an event

-The base class has not been changed in order to add new boss features

✅ Inheritance,✅ Polymorphism,✅ Event-Driven Design,✅ Open/Closed Principle
